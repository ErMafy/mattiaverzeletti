🚀 DEPLOY SU VERCEL (3 OPZIONI):

OPZIONE A: GitHub Push
───────────
1. git init
2. git add .
3. git commit -m "Initial commit"
4. git push origin main
5. Vai su vercel.com → connect repo

OPZIONE B: Vercel CLI
────────────
1. npm i -g vercel
2. vercel
   (segui le istruzioni)

OPZIONE C: Dashboard Vercel
──────────────
1. Vai su vercel.com
2. Click "New Project"
3. Import il repository
4. Click "Deploy"
